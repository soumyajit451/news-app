# news-app
Gives you news


## How to develop

```bash
# clone the repo
git clone https://github.com/tikendraw/news-app.git
cd news-app

# create venv
python3 -m venv .venv

# activate it
source .venv/bin/activate

# install dependencies
pip install -r requirements_dev.txt

#install pre-commit
pre-commit install
```
